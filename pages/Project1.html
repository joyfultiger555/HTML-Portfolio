Personas and Storyboarding

For this project we were attempting to enter the mind of a user. By understanding the user's thought process while using an interface that could give a better idea of what needs to be improved or what is working. I chose a self service checkout machine and observed people using the interface in order to develop patters from these patterns i was then able to create user stories as well as personas in order to fully encompass the user's mindset and usage process of the interface.
